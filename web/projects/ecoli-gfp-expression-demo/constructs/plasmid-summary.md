# Plasmid Design Summary

## Construct ID

- `pGLO`

## Intended Architecture

1. PBAD arabinose-inducible promoter
2. RBS (strong bacterial consensus)
3. sfGFP coding sequence
4. Double stop codon
5. Terminator
6. Ampicillin resistance marker (bla)
7. ColE1-like origin with AraC regulatory module

## Host and Selection

- Host: E. coli K-12 HB101
- Selection: Ampicillin 100 ug/mL

## Verification Targets

- Colony PCR amplicon matching expected insert length
- Sanger confirmation of PBAD-RBS-GFP junctions
- No frame-shift in GFP CDS
